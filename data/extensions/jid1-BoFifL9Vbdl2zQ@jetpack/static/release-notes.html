Decentraleyes v1.2.2



NEW: Get a first glimpse of what this add-on actually does for you! Navigate to the main menu and click "Add-ons" > "Decentraleyes" > "Preferences" to get familiar with the new "local injections" counter. Please note: it's new and will start from zero (ignoring prior injections).

NEW: Support for Fennec (or Firefox for mobile). If you're running Firefox on your smartphone, then you might be happy to know that this is the very first release to be fully compatible with mobile versions of Firefox. Take this add-on with you to protect yourself on-the-go!

EXPERIMENTAL: Whitelist specific domains to force Decentraleyes to ignore them. Navigate to the main menu and click "Add-ons" > "Decentraleyes" > "Preferences". From there, you can add domains to the "whitelist" field. This might come in handy if you run into issues with a certain website.


NOTE: Decentraleyes works right out of the box and no manual configuration is required! You can check if you're properly protected by using the testing utility.


General information:

Decentraleyes is a very young project, so expect many improvements in the realms of clear, present, user feedback and ease-of-use as it matures. It does a lot to protect you, but is a tad shy about it.


Detailed changelog:
  • Decentraleyes now supports Fennec (better known as Firefox for mobile).
  • Experimental support for whitelisting domains from preferences (Add-ons Manager).
  • Amount of injected resources shown in preferences (Add-ons Manager).
  • Fixed a rare bug where shorthand mapping notations broke specific websites.
  • Removed Pale Moon compatibility as it will receive separate, tailored, builds.
  • Took out rare resources to make room for more common candidates.
  • Added new AngularJS, Dojo, and jQuery UI scripts to the resource bundle.
